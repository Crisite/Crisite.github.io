linux分区扩容

/home分了100G都满了，急需扩容

本机分完之后是这个样子，/dev/nvme0n1p15原本是windows下的E盘，删除卷后有一个199G的空白分区，但是fdisk识别不到这个空白分区

于是先下载一个gparted

sudo pacman -S gparted

打开gparted后将空白分区新建为ext4格式的分区

然后就可以挂载了

sudo vim /etc/fstab

# 在文件中添加
/dev/nvme0n1p15 /home/crisite/disk ext4 defaults 0 1

其中home/crisite/disk是挂载目录，本来disk文件夹不存在，重启或者mkdir即可

可能更好的解决方法，通过逻辑卷的方式进行扩容，画饼吧，以后再做

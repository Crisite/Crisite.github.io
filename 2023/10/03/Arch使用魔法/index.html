Arch使用魔法

博客部署好之后就该让访问博客变得轻松了 TvT

1. 下载clash for windows

yay -S clash-for-windows-bin

终端输入cfw即可启动clash fow windows

界面简介

  • General（常规）：
  • Port、Socks Port；分别为 HTTP、SOCKS 代理端口，点击终端图案可以打开一个配置了代理的命令行窗口，点击端口数字可以复制该命令；
  • Allow LAN：启用局域网共享代理功能；
  • Log Level：日志等级；
  • Home Directory：点击下方路径直达 ~/.config/clash 文件夹；
  • GeoIP Database：点击下方日期可更新 GeoIP 数据库；
  • UWP Loopback ：可以用来使 UWP 应用解除回环代理限制；
  • Tap Device ：安装 cfw-tap 网卡，可用于处理不遵循系统代理的软件（实际启动 tap 模式需要更改配置文件）；
  • General YML：编辑 config.yml 文件，可用于配置部分 General 页面内容；
  • Dark Theme：控制暗色模式；
  • Start with linux：设置开机自启；
  • Proxies（代理）：选择代理方式（Global – 全局、Rule – 规则、Direct – 直连）及策略组节点选择；
  • Profiles（配置管理）：
  • 用来下载远端配置文件和创建本地副本，且可在多个配置文件间切换；
  • 对配置进行节点、策略组和规则的管理（添加节点、策略组和规则在各自编辑界面选择 Add, 调整策略组顺序、节点顺序及策略组节点使用拖拽的方式）；
  • Logs（日志）：显示当前请求命中规则类型和策略；
  • Connections (连接): 显示当前的 TCP 连接，可对某个具体连接执行关闭操作；
  • Settings（设置）：软件详细设置；
  • Feedback（反馈）：显示软件、作者相关信息。

使用教程

  • 远程订阅地址

远程订阅地址即通过 URL 链接导入，点击界面左侧菜单 Profiles，在顶部输入框填入 URL 连接地址并点击 Download 即可，下载完成后点击对应的配置文件即可添加配置文件

个人使用的代理->[SSRDog][https://dog.ssrdog111.com/#/register?code=UNM89GSi]

  • 设置本机代理

export

export https_proxy=http://127.0.0.1:%mixedPort%
export http_proxy=http://127.0.0.1:%mixedPort%
export all_proxy=socks5://127.0.0.1:%mixedPort%

不同的终端命令所识别的环境变量名不同，如all_proxy对curl生效，而对wegt则不生效

！ 但是设置当前终端代理方式后，当前终端通过curl ip.gs查看ip地址时，修改生效，但是仍然无法ping通Google，于是切换全局代理

在/etc/environment中添加代理

https_proxy=http://127.0.0.1:%mixedPort%
http_proxy=http://127.0.0.1:%mixedPort%
all_proxy=socks5://127.0.0.1:%mixedPort%

然后重启重新打开clash for windows，在浏览器中访问Google，发现成功访问Google：

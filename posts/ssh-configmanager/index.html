SSH配置管理

  1. 执行生成ssh密钥命令生成第二个密钥生成 RSA 密钥对，但在接下来的选项中指定一个不同的文件名，例如：
ssh-keygen -t rsa -c "your@email.com"
Enter file in which to save the key (/Users/userName/.ssh/id_rsa): /Users/userName/.ssh/github_id_id_rsa
  1. 在 .ssh 目录下添加配置文件
# gitlab
Host ip
HostName ip
User yourname
PreferredAuthentications publickey
IdentityFile C:\Users\username\.ssh\id_rsa

# github
Host github.com
HostName github.com
User crisite
PreferredAuthentications publickey
IdentityFile C:\Users\username\.ssh\github_id-rsa

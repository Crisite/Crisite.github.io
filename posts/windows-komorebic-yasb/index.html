Windows+Komorebic+Yasb

星期一，七月 29 日，2024 年 | 1 分钟阅读 | 更新于 星期一，七月 29 日，2024 年

泡芙

使用 Komorebic+Yasb 在 Windows 中也获得类似 dwm 的平铺式窗口的能力

下载 Komorebic 和 Yasb

# komorebi
winget install LGUG2Z.komorebi
winget install LGUG2Z.whkd

git@github.com:da-rth/yasb.git
cd yasb
# 事实上因为版本不一致, 我 pip 一个一个下的
pip install -r requirements.txt

mkdir C:/Users/{username}/.yasb
cp ./style.css C:/Users/{username}/.yasb/
cp ./config.yaml C:/Users/{username}/.yasb/

# 此处还是会有报错, 因为 git 仓库的配置文件缩进有问题 (也是离谱)
python src/main.py

参考: pull requests

接下来还可能碰到的问题是 图标无法正常显示 解决方法: resolve action

另外, komorebi 可能还需要一些配置优化, 那么需要注意的是

{
  "default_workspace_padding": 2,
  "default_container_padding": 2,
  "border_width": 2,
  "border_offset": 2
}

其中, 这四个属性是无法热加载的, 必须要重启应用

开机自启动

win + r 打开运行, 输入 shell:startup 就可以打开程序启动的路径了, 在这个路径下的bat脚本会开机自启动 创建一个 dwm.bat

@echo off
komorebic start --whkd
start "" "pythonw" "D:\yasb\src\main.py"
